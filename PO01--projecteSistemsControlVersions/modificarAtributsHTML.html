Modificar Atributs HTML

JavaScript com a un dels millors llengutatges i més utilitzats te la possiblitat de modificar els atributs dels elements HTML. Aquesta funcionalitat ens permet anar adaptant els elements segons la funcionalitat del nostre lloc web, com:

  • Canviant valors d´un atribut.
  • Afegint un atribut si no existeix (afegir també els seus valors).
  • Obtenint el valor actual de l´atribut.
  • Etc.

A l´hora de treballar amb els elements, JavaScript ja disposa de funcions encarregades de poder modificar/alterar l´estat, el valor i demés dels atributs Quan treballem amb formularis ens ve de perles poder obtenir tota aquesta informació per traslladar-li-la a la BBDD.

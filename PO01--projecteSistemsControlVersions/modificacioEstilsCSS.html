Modificar Estils CSS

JavaScript com a un dels millors llengutatges i més utilitzats te la possiblitat de modificar els estils dels elements CSS. Aquesta funcionalitat ens permet anar adaptant els elements segons la funcionalitat del nostre lloc web, com:

  • Canviant les classes d´un element.
  • Afegint classes les quals fasen fer coses o no al element.
  • Ocultar i/o mostrar elements a la web.
  • Etc.

A l´hora de treballar amb els elements, JavaScript ja disposa de funcions encarregades de poder modificar/alterar l´estat, el valor i demés valos CSS.
